# 第二幕：出击 —— 像CEO一样配置资源

> 心法锚点：斯多葛的控制二分法 —— 在资源匮乏时保持战略定力，不因焦虑而乱投资。

本幕任务：定方向和攒弹药。从战略到财务，学会把有限资源用在回报最高的地方。

---

## 第05课：把自己当产品——SWOT与MVP（商业分析）

### 核心理论

把你自己当成一款产品来经营。SWOT分析找定位，MVP思维做验证。

**SWOT个人分析**：
- **Strengths（优势）**：你比同龄人做得更好的事是什么？不一定是"特长"，可能是"别人觉得难但你觉得简单"的事
- **Weaknesses（劣势）**：诚实面对短板，但区分"可以提升的"和"不值得补的"
- **Opportunities（机会）**：正在增长的行业、正在出现的岗位、你身边没人做但有需求的事
- **Threats（威胁）**：AI替代风险、行业萎缩、政策变化

**MVP思维**：不要闷头打磨三年才"上线"。用最小成本验证你的假设：
- 想转行做产品经理？先花一个月写三篇产品分析，发在社交媒体上，看有没有反馈
- 想创业？先用一个周末做一个最简版本，找10个人试用
- 想出国？先上一门目标学校的公开课，看自己是否真的感兴趣

### 决策卡：MVP试错清单
1. 我要验证的假设是什么？
2. 最小成本的验证方式是什么？
3. 多久能看到结果？
4. 什么结果算验证通过？什么算失败？

### 现实案例
"要不要转行做产品经理"：一个在传统行业干了三年的年轻人，与其辞职报培训班，不如先用MVP试错——写产品分析、参加行业社群、找产品经理聊天，用三个月低成本验证自己是否真的适合，而不是赌上全部身家。

## 第06课：选择大于努力的数学证明（经济学+概率论）

### 核心理论

**赛道供需决定价值**：你再努力，如果赛道供给过剩，你的市场价格就是低的。这不是你的问题，是市场结构的问题。

**沉没成本**：过去花掉的时间和钱，已经花了，不应该影响你未来的决策。唯一要问的是："从今天起，继续还是换方向，哪个期望收益更高？"

**贝叶斯定理（人话版）**：每收到一条新信息，就更新一次你的判断，别死守第一印象。

### 嵌入心法：斯多葛控制二分法
讲沉没成本时直接引入：你能控制的是下一步怎么走，不能控制的是过去已经花掉的时间和钱。止损不是失败，是理性。

### 赛道评分表（五维度）

对你当前或考虑的方向，每项打1-10分：

| 维度 | 问题 | 评分 |
|------|------|------|
| 需求增长 | 这个方向的市场需求在增长还是萎缩？ | /10 |
| 支付意愿 | 市场愿意为这个方向的能力付多少钱？ | /10 |
| 进入门槛 | 新人进入有多难？（门槛越高对已入场者越有利） | /10 |
| 可迁移性 | 这个方向的技能能迁移到其他领域吗？ | /10 |
| 上限空间 | 做到顶尖，天花板有多高？ | /10 |

**总分低于30分**：认真考虑换赛道。

### 止损触发器清单
提前设定止损条件，写下来：
- "如果X个月内没有达到Y，我就……"
- "如果又出现Z这种情况，我就……"

关键：止损条件必须在你清醒的时候设定，不是在你痛苦的时候临时决定。

### 信息更新日志模板
| 日期 | 新增证据 | 我的判断怎么变 | 下一步要验证什么 |
|------|---------|---------------|----------------|
| | | | |

### 现实案例
"考研考到第三年要不要继续"：已经花的两年是沉没成本。要问的是：从今天起，再花一年考研的期望收益，vs 用这一年找工作积累经验的期望收益，哪个更高？如果你设定的止损条件（比如"三次不过就转方向"）已经触发了，执行它。

## 第07课：复利与折现——时间是你唯一的原始资本（金融学）

### 核心理论

**复利思维**：年轻时的每一小时投入，都是在决定十年后你的价值。关键不是"有没有在吃苦"，而是"这个苦能不能产生积累"。

两种努力的十年对比：
```
复利型努力：技能积累、作品集、信用构建
年1: 1 → 年2: 1.2 → 年3: 1.44 → … → 年10: 6.19

消耗型努力：简单重复、不产生积累
年1: 1 → 年2: 1 → 年3: 1 → … → 年10: 1
```

### 决策卡：我的时间在复利还是在消耗？
判断清单：
- 这项工作一年后会让我更值钱吗？（是→复利，否→消耗）
- 我在积累可展示的成果吗？（作品集、案例、人脉）
- 如果我明天离开现在的岗位，我能带走什么？

### 现实案例
"每天加班到博但学不到东西"：一个在大厂做螺丝钉的年轻人。加班不是问题，问题是这个班加完你的市场价值有没有增加。如果只是重复性劳动，你需要的不是更努力，而是换一个能让你积累复利的位置。

## 第08课：信用与负债——如何撬动你没有的资源（金融学）

### 核心理论

**良性负债vs恶性负债**：
- 良性负债：能帮你赚更多钱的负债（教育贷款投资高回报技能、创业初期的合理融资）
- 恶性负债：只满足当下欲望的负债（消费贷买奢侈品、分期买超出预算的手机）

简单判断："往你口袋里塞钱的是资产，从你口袋里掏钱的是负债。"

**信用是隐形资产**：对普通家庭子弟来说，信用（靠谱、守信、能交付）可能是最容易被忽视但最有价值的资本。它不需要花钱积累，但需要很长时间构建，毁掉只需要一次。

### 决策卡：借债决策三问
1. 这笔钱能生钱吗？（能→考虑，不能→警惕）
2. 我还不上怎么办？（有Plan B→可以，没有→不借）
3. 不借会怎样？（没有不借的选项→说明你已经在被动了，先解决根本问题）

### 现实案例
"要不要借钱报培训班/买房"：用良性/恶性负债框架分析。培训班：如果这个技能能在一年内让你的收入增长超过还贷成本，是良性负债；如果只是缓解"我在做事"的焦虑，是恶性负债。买房：如果房贷占收入比例合理且有稳定还款能力，是良性；如果掏空六个钱包且月供吃力，是恶性。
