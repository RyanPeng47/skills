# 人生破局 (pojiu)

> 给普通家庭子弟的决策方法论。用经济学的脑做决策，用社会学的眼看局势，用哲学的心定输赢。

一套将**经济学、博弈论、社会学、心理学、哲学**融合为可操作工具的人生决策 Skill。

不灌鸡汤，不说"加油"，用严肃的学科工具帮你拆解真实的人生问题。

## 这是什么

中国有太多聪明、努力的年轻人，不是因为不够拼，而是因为缺少**做选择的方法论**和**信息源**。

做选择是一个需要系统训练的能力，不是靠"想开点"就能解决的。

这套 Skill 把一整套人生决策方法论（15课完整体系）封装成 AI 工具，让每一个普通学子都能在选择这件事上少走弯路。

**致敬所有为普通人弥补信息差而努力过的人。**

## 能帮你什么

| 你的困惑 | Skill 做什么 |
|---------|------------|
| 考研还是工作？要不要转行？ | 赛道评分表 + 沉没成本分析 + 贝叶斯更新 |
| 跟父母意见不合 | 阿德勒课题分离 + 三段式沟通脚本 |
| 职场被欺负 | 博弈类型矩阵 + 以牙还牙策略 |
| 努力但没结果 | 复利vs消耗判断 + 杠杆点寻找 |
| 焦虑迷茫 | 斯多葛控制二分法 + 庄子无用之用 |
| 不知道自己想要什么 | 布迪厄文化资本分析 + 尼采精神三变 |

## 理论框架

**"剑、盾、心"三位一体：**

- ⚔️ **进攻之剑**：经济学、金融学、概率论、商业分析
- 🛡️ **防御之盾**：博弈论、社会学（布迪厄/格兰诺维特）、系统思维
- ❤️ **压舱之心**：脑科学、阿德勒心理学、尼采哲学、斯多葛学派、庄子哲学

## 安装

### Claude Code

```bash
# 推荐：一行安装
git clone https://github.com/[your-username]/pojiu.git /tmp/pojiu && cp -r /tmp/pojiu/skills/pojiu ~/.claude/skills/ && rm -rf /tmp/pojiu
```

### 手动安装

将 `skills/pojiu/` 文件夹复制到你的 Claude Code skills 目录：

```bash
cp -r skills/pojiu ~/.claude/skills/
```

## 使用方式

在 Claude Code 中直接描述你的困惑即可。Skill 会自动识别并调用对应的框架：

```
你：我在纠结要不要考研，已经准备了两年了但一直没考上。

Claude：[调用赛道评分表 + 沉没成本分析]
       [给出诊断 + 决策卡 + 行动建议]
```

## 课程结构

```
第一幕：觉醒（01-04课）── 看清你站在哪里
  │  心法锚点：尼采精神三变
  │
第二幕：出击（05-08课）── 像CEO一样配置资源
  │  心法锚点：斯多葛控制二分法
  │
第三幕：交手（09-12课）── 在人群中生存和借力
  │  心法锚点：庄子游刃有余
  │
第四幕：扎根（13-15课）── 定义属于你的好生活
     心法锚点：尼采爱命运
```

## 每次对话的标准交付

每次使用 Skill 帮你分析问题时，你会拿到：

1. **一个框架图**：核心模型的可视化
2. **一张决策卡**：明天就能用的行动指南
3. **一个现实案例**：普通家庭子弟的典型场景
4. **本次一句话**：核心认知浓缩
5. **48小时行动**：15分钟内可完成的具体下一步

## 适用人群

- **核心受众**：22-30岁，已经或即将进入职场的普通家庭子弟
- **同样适用**：12-18岁需要提前建立决策思维的学生、30-50岁想帮孩子（也帮自己）的父母
- **一句话**：所有觉得"我很努力但不知道该怎么选"的人
